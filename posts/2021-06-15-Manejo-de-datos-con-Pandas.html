1. Resumen¶

Vamos a ver una pequea introduccin a la librera de manipulacin y analisis de datos Pandas. Con ella podremos manejar y procesar datos tabulares que nos ayudar para poder operar con ellos y obtener informacin de una manera muy valiosa

2. ¿Qué es Pandas?¶

Pandas es una librera de Python que est diseada para que el trabajo con datos relacionales o etiquetados sea fcil e intuitivo

Pandas est diseado para muchos tipos diferentes de datos:

  • Datos tabulares con columnas de tipos heterogneos, como en una tabla SQL o una hoja de clculo de Excel
  • Datos de series de tiempo ordenados y desordenados (no necesariamente de frecuencia fija).
  • Datos matriciales arbitrarios (homogneos o heterogneos) con etiquetas de fila y columna
  • Cualquier otra forma de conjuntos de datos observacionales/estadsticos. No es necesario etiquetar los datos en absoluto para colocarlos en una estructura de datos de pandas.

Las dos estructuras de datos principales de Pandas son las Series (unidimensional) y los DataFrames (bidimensional). Pandas est construido sobre NumPy y est destinado a integrarse bien dentro de un entorno informtico cientfico con muchas otras bibliotecas de terceros.

Para los cientficos de datos, el trabajo con datos generalmente se divide en varias etapas: recopilar y limpiar datos, analizarlos/modelarlos y luego organizar los resultados del anlisis en una forma adecuada para trazarlos o mostrarlos en forma de tabla. Pandas es la herramienta ideal para todas estas tareas.

Otra caracterstica es que pandas es rpido, muchos de los algoritmos de bajo nivel se han construido en C

En Pandas existen dos tipos de estructuras de datos: las Series y los DataFrames

3.1. Series¶

El tipo de dato Serie es una matriz etiquetada unidimensional capaz de contener cualquier tipo de datos (enteros, cadenas, nmeros de punto flotante, objetos Python, etc.). Estn divididas en ndices.

Para crear un tipo de dato Serie la forma ms comn es

serie = pd.Series(data, index=index)

Donde data puede ser

  • Un diccionario
  • Una lista o tupla
  • Un ndarray de Numpy
  • Un valor escalar

Como uno de los tipos de datos puede ser un ndarray de Numpy, importamos Numpy para poder usarlo

3.2. DataFrames¶

Un DataFrame es una estructura de datos etiquetada y bidimensional, con columnas de tipos potencialmente diferentes, es decir, en una columna puede haber datos de tipo entero, en otra columna datos de tipo string, etc. Puede pensar en ello como una hoja de clculo o una tabla SQL, o un diccionario de objetos Series.

Es el objeto pandas ms utilizado. Al igual que las Series, los DataFrames aceptan muchos tipos diferentes de entrada:

Junto con los datos, opcionalmente puede pasar argumentos de ndice (etiquetas de fila) y columnas (etiquetas de columna). Si pasa un ndice y/o columnas, est garantizando el ndice y/o columnas del DataFrame resultante. Por lo tanto, un diccionario de Series ms un ndice especfico descartar todos los datos que no coincidan con el ndice pasado

Si no se pasan las etiquetas de los ejes, se construirn a partir de los datos de entrada basndose en reglas de sentido comn.

12.1.1. Intersección de concatenación¶

Hay dos maneras de hacer la concatenacin, cogiendo todos los ndices de los DataFrames o cogiendo solo los que coinciden, esto se determina mediante la variable join, que admite los valores 'outer' (por defecto) (coge todos los ndices) o 'inner' (solo los que coinciden)

Veamos un ejemplo de 'outer'

In [ ]:
dataframe1 = pd.DataFrame(
    {
        "A": ["A0", "A1", "A2", "A3"],
        "B": ["B0", "B1", "B2", "B3"],
        "C": ["C0", "C1", "C2", "C3"],
        "D": ["D0", "D1", "D2", "D3"],
     },
     index=[0, 1, 2, 3])

dataframe4 = pd.DataFrame(
    {
        "B": ["B2", "B3", "B6", "B7"],
        "D": ["D2", "D3", "D6", "D7"],
        "F": ["F2", "F3", "F6", "F7"],
     },index=[2, 3, 6, 7])

diccionario = {

dataframe.dropna(how="any")

uno dos
