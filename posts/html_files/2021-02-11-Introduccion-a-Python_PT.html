Code:

int

2.3.1. Listas

Listas armazenam vários itens em uma variável. Eles são declarados utilizando os símbolos [], com itens separados por vírgulas.

Code:
lista = ['item0', 'item1', 'item2', 'item3', 'item4', 'item5']


lista
Output:
['item0', 'item1', 'item2', 'item3', 'item4', 'item5']

Podemos obter a extensão de uma lista utilizando a função 'len()len().

Code:
len(lista)
Output:
6

As listas podem ter itens de diferentes tipos

Code:
lista = ['item0', 1, True, 5.3, "item4", 5, 6.6]


lista
Output:
['item0', 1, True, 5.3, 'item4', 5, 6.6]

Em Python começamos a contar a partir da posição 0, ou seja, se quisermos obter a primeira posição na lista

Code:
lista[0]
Output:
'item0'

Mas uma das coisas poderosas sobre Python é que, se quisermos acessar a última posição, podemos usar índices negativos.

Code:
lista[-1]
Output:
6.6

Se ao invés da última posição na lista, queremos a penúltima

Code:
lista[-2]
Output:
5

Se quisermos apenas um intervalo de valores, por exemplo, do segundo ao quinto item, utilizamos [2:5].

Code:
lista[2:5]
Output:
[True, 5.3, 'item4']

Se o primeiro número do intervalo for omitido, significa que queremos ir do primeiro item da lista para o item indicado, ou seja, se quisermos ir do primeiro item para o quinto item, utilizamos [:5].

Code:
lista[:5]
Output:
['item0', 1, True, 5.3, 'item4']

Se o último número do intervalo for omitido, significa que queremos do item indicado para o último, ou seja, se quisermos do terceiro item para o último que utilizamos [3:].

Code:
lista[3:]
Output:
[5.3, 'item4', 5, 6.6]

Você pode escolher a gama de itens também com números negativos, ou seja, se quiser desde a antepenúltima até a penúltima que você utiliza [-3:-1]. Isto é útil quando você tem listas que não sabe seu comprimento, mas você sabe que quer uma faixa de valores a partir do final, porque, por exemplo, a lista foi criada com medidas que você está tomando e você quer saber as últimas médias.

Code:
lista[-3:-1]
Output:
['item4', 5]

Você pode verificar se um item está na lista por

Code:
'item4' in lista
Output:
True

2.3.1.2. Compreensão da lista

Podemos operar através da lista

Code:
fruits = ["apple", "banana", "cherry", "kiwi", "mango"]

lista2

2.3.2. Duplo

Os tuplos são semelhantes a listas, eles armazenam vários itens em uma variável, podem conter itens de diferentes tipos, mas não podem ser modificados ou reordenados. Eles são definidos por (), com itens separados por vírgulas.

Ser inalterável faz com que os tuplos funcionem um pouco mais rápido que as listas, portanto, se você não precisar modificar os dados, é melhor usar os tuplos do que as listas.

Code:
tupla = ('item0', 1, True, 3.3, 'item4', True)


tupla
Output:
('item0', 1, True, 3.3, 'item4', True)

Seu comprimento pode ser obtido utilizando a função len().

Code:
len (tupla)
Output:
6

Para criar tuplos com um único elemento, é necessário acrescentar uma vírgula

Code:
tupla = ('item0',)


tupla, type(tupla)
Output:
(('item0',), tuple)

Para acessar um elemento tuple, proceda da mesma forma que para as listas.

Code:
tupla = ('item0', 1, True, 3.3, 'item4', True)


print(tupla[0])


print(tupla[-1])


print(tupla[2:4])


print(tupla[-4:-2])
Output:
item0
True
(True, 3.3)
(True, 3.3)

Podemos verificar se há um item no tuple

Code:
'item4' in tupla
Output:
True

Code:
lista = list(tupla)


lista[4] = 'ITEM4'


tupla = tuple(lista)


tupla

item0, item1, item2, item5
Output:

7.1. Inheritance

A herança nos permite definir uma classe que herda todos os métodos e propriedades de outra classe.

A classe pai é a classe da qual ela é herdada, também chamada classe base*.

A classe infantil é a classe que herda de outra classe, também chamada classe derivada*.

Criamos uma classe mãe

Code:
class Persona:


  def __init__(self, nombre, apellido):


    self.nombre = nombre


    self.apellido = apellido




  def imprimir_nombre(self):


    print(f'Me llamo {self.nombre} {self.apellido}')




objeto_padre = Persona("Laura", "Perez")


objeto_padre.imprimir_nombre()
Output:
Me llamo Laura Perez

Para criar a classe criança, é necessário indicar entre parênteses, ao declarar a classe, de que classe ela herda.

Code:
class Estudiante(Persona):


  pass

E ao criar o objeto da classe criança, os parâmetros que a classe pai precisa são passados para ela.

Code:
objeto_hijo = Estudiante("Mariano", "Sanz")


objeto_hijo.imprimir_nombre()
Output:
Me llamo Mariano Sanz

Até agora a classe infantil herdou as funções da classe dos pais, mas nós podemos modificá-las reescrevendo-as. Por exemplo, reescrevendo a função duder init.

Se você reescrever a função dunder init, se você quiser que a função dunder init da classe mãe seja chamada, você tem que chamá-la.

Há duas maneiras de fazer isso, uma é através do nome da classe mãe, neste caso você tem que passar-lhe a variável "si mesmo".

Code:
class Estudiante(Persona):


  def __init__(self, nombre, apellido):


    Persona.__init__(self, nombre, apellido)




objeto_hijo = Estudiante("Mariano", "Sanz")


objeto_hijo.imprimir_nombre()
Output:
Me llamo Mariano Sanz

Outra forma é através do super(), neste caso você não precisa passar a variável self.

Code:
class Estudiante(Persona):


  def __init__(self, nombre, apellido):


    super().__init__(nombre, apellido)




objeto_hijo = Estudiante("Mariano", "Sanz")


objeto_hijo.imprimir_nombre()
Output:
Me llamo Mariano Sanz

7.2. Sobrecarga do operador

Podemos definir operações básicas, como a adição, entre vários objetos de uma classe. Por exemplo, se tivermos uma classe representando um vetor, podemos definir adição e multiplicação entre os objetos dessa classe

Code:
class Vector:


    def __init__(self, x, y):


        self.x = x


        self.y = y


    


    def __add__(self, other):


        return Vector(self.x + other.x, self.y + other.y)


    


    def __mul__(self, other):


        return Vector(self.x * other.x, self.y * other.y)


    


    def __str__(self):


        return f"Vector ({self.x}, {self.y})"




v1 = Vector(1, 2)


v2 = Vector(3, 4)


print(v1 + v2)  # Vector (4, 6)


print(v1 * v2)  # Vector (3, 8)
Output:
Vector (4, 6)
Vector (3, 8)

class Numeros:


  def __iter__(self):


    self.a = 1


    return self




  def __next__(self):


    x = self.a


    self.a += 1


    return x




objeto_iterador = Numeros()


iterador = iter(objeto_iterador)




print(next(iterador), end=' ')


print(next(iterador), end=' ')


print(next(iterador), end=' ')


print(next(iterador), end=' ')


print(next(iterador), end=' ')

O exemplo acima continuaria para sempre se você tivesse chamadas suficientes para o seguinte(), ou se fosse utilizado em um loop `para'.

Para evitar que a iteração continue para sempre, podemos utilizar a declaração 'StopIteration'.

No método __next__(), podemos adicionar uma condição de terminação para gerar um erro se a iteração for realizada um número especificado de vezes:

class Numeros:


  def __iter__(self):


    self.a = 1


    return self




  def __next__(self):


    if self.a <= 20:


      x = self.a


      self.a += 1


      return x


    else:


      raise StopIteration




objeto_iterador = Numeros()


iterador = iter(objeto_iterador)




for x in iterador:


  print(x, end=' ')

9. escopo das variáveis

Uma variável só está disponível dentro da região em que é criada. Isto é chamado de âmbito.

9.1. Alcance local

Uma variável criada dentro de uma função pertence ao escopo local dessa função e só pode ser usada dentro dessa função.

Code:
def funcion():


  x = 300


  print(x)




funcion()
Output:
300

A variável x não está disponível fora da função, mas está disponível para qualquer função dentro da função.

Code:
def funcion():


  x = 300


  def funcion_interna():


    print(x)


  funcion_interna()




funcion()
Output:
300

9.2. Alcance global

Uma variável criada no corpo principal do código Python é uma variável global e pertence ao escopo global.

As variáveis globais estão disponíveis a partir de qualquer escopo, global e local.

Code:
x = 300




def funcion():


  print(f'Ámbito local: {x}')




funcion()




print(f'Ámbito global: {x}')
Output:
Ámbito local: 300
Ámbito global: 300

Se você criar duas variáveis, uma global e uma local, ambas com o mesmo nome, Python as criará como duas variáveis diferentes.

Code:
x = 300




def funcion():


  x = 200


  print(f'Variable local: {x}')




funcion()




print(f'Variable global: {x}')
Output:
Variable local: 200
Variable global: 300

Se você precisa criar uma variável global, mas ela é declarada localmente, você pode utilizar a palavra-chave global.

A palavra-chave global torna a variável global.

Code:
def funcion():


  global x


  x = 300




funcion()




print(f'Variable global: {x}')
Output:
Variable global: 300

Além disso, o uso da palavra-chave "global" faz uma mudança para uma variável global dentro de uma função.

Code:
x = 300




def funcion():


  global x


  x = 200




funcion()




print(f'Variable global: {x}')
Output:
Variable global: 200

12. Tente... Exceto

Quando um erro ocorre, ou uma exceção como é realmente chamada, Python normalmente pára e gera uma mensagem de erro.

Essas exceções podem ser tratadas utilizando as declarações de "tentativa" e "exceção".

Code:
try:


  print(variable_no_declarada)


except:


  print("Ha ocurrido una excepción")
Output:
Ha ocurrido una excepción

Como o bloco "tentativa" gera um erro, então o bloco "exceção" será executado.

Sem o bloco "tentativa", o programa travaria e geraria um erro.

Você pode definir quantos blocos de exceção quiser, por exemplo, se quiser executar um bloco de código especial para um tipo especial de erro

Code:
try:


  print(variable_no_declarada)


except NameError:


  print("La variable \'variable_no_declarada\' no está definida")


except:


  print("Algo inesperado ha ocurrido")
Output:
La variable 'variable_no_declarada' no está definida

A palavra 'else' pode ser utilizada para indicar o caso em que não ocorreu um erro.

Code:
try:


  print('MaximoFN')


except NameError:


  print("Ha ocurrido una excepción")


else:


  print('Todo OK')
Output:
MaximoFN
Todo OK

com a palavra finalmente um código será executado no final, quer uma exceção tenha ocorrido ou não.

Code:
try:


  print(variable_no_declarada)


except:


  print("Ha ocurrido una excepción")


finally:


  print("'try except' finallizado")
Output:
Ha ocurrido una excepción
'try except' finallizado

Isto pode ser útil para fechar objetos e recursos de limpeza.

Code:
class Clase:


  variable = 'MaximoFN'




objeto = Clase()




try:


  print(Clase.mi_variable)


except:


  print("Ha ocurrido una excepción")


finally:


  del objeto
Output:
Ha ocurrido una excepción
