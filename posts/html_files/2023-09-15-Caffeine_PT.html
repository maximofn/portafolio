Cafeína¶

Aviso: Este post foi traduzido para o português usando um modelo de tradução automática. Por favor, me avise se encontrar algum erro.

Caffeine é um software para evitar que a tela do nosso Linux seja bloqueada e depois entre em modo de suspensão. Para isso, forneceremos cafeína.

Instalação¶

A instalação é muito simples, basta fazer

sudo apt update
sudo apt install caffeine

Cafeína infinita¶

Se pressionarmos o botão windows do teclado e escrevermos caffeine-indicator, aparecerá a opção de iniciar o indicador do software.

Se executarmos, aparecerá um indicador do software na barra superior do Ubuntu

Se clicarmos no indicador, aparecerá um menu.

Agora podemos clicar em Activate para que nosso Ubuntu tenha cafeína infinita e nunca durma.

Cafeína para um comando¶

Uma coisa que me acontece é que quando estou treinando uma rede neural, gosto de deixar a avaliação do modelo visível, mesmo que eu não esteja no computador. Mas se esse treinamento durar horas, em algum momento o protetor de tela é ativado e já não se vê mais a evolução do treinamento, então para resolver isso temos caffeinate pelo qual fazendo

`caffeinate COMMAND`

Faremos com que nosso computador tenha cafeína e não durma enquanto o comando COMMAND está sendo executado.
