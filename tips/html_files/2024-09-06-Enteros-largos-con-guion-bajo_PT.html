Números inteiros longos com sublinhado¶

Aviso: Este post foi traduzido para o português usando um modelo de tradução automática. Por favor, me avise se encontrar algum erro.

Se tivermos um número inteiro muito longo, pode ser difícil lê-lo

num = 1000000000000
print(num)

1000000000000

Mas o Python nos permite colocar sublinhados para separar os dígitos e torná-los mais legíveis.

num_ = 1_000_000_000_000
print(num_)

1000000000000

Vamos verificar se ambos números são iguais.

num == num_
Out[4]:
True
